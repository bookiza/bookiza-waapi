Welcome to Bookiza's official documentation.

Bookiza is a book reification framework for the web. It cuts down book writing and publishing time by half.

By half, no less!

Manage your manuscript smartly and bake-in all the goodness of the web into your book. Write books that work everywhere.

Note: Bookiza uses the terminal environment (shell) and NodeJS. If you're uncomfortable about using shell commands or uncertain about what the terminal app or NodeJS is, please consider using Bubblin's "in-browser" editor instead.
