Keyboard shortcuts:

KEYS ACTION
← / → or Spacebar: Previous / Next page
⌘ + ←: Start of book
⌘ + →: End of book
⌘ + U: Open menu
⌘ + M: New manuscript
⌘ + S: Save changes
⌘ + P: Publish book
⌘ + K: See shortcuts
⌘ + E: Go home
⌘ + J: Search
⌘ + L: Login / logout

Shortcuts work only on desktop & tablet browsers when a hardware keyboard is attached.
